# CodeAlpha Data Analytics Internship

This repository contains the tasks completed as part of the CodeAlpha Data Analytics Internship.

## Tasks Included

### Task 2 – Stock Price Analysis
Exploratory Data Analysis (EDA) performed on historical stock price data using Python, Pandas, and data visualization techniques to identify trends and patterns.

### Task 3 – IPL Player Performance Analysis
Analysis of IPL player performance using real match data, including data visualization and performance insights.
